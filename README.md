# HealthyBite — Prototype Website
Author: Vaidehee Patel

Description: Responsive meal planning and nutrition website prototype for IT499 Capstone.

Includes: Home, Recipes, Meal Planner, BMI Calculator, Contact pages. No backend required — uses browser storage for meal plans.

How to run locally: Open index.html in a web browser. To deploy to GitHub Pages, push the files to a repo and enable Pages from the main branch.
